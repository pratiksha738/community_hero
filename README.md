## 🌍 Overview

**Community Hero App** is a full-stack civic engagement platform that enables citizens to report local infrastructure issues like potholes, garbage dumps, broken streetlights, and water leakage using images and geolocation.

It helps communities and local authorities track, visualize, and resolve issues efficiently through a centralized dashboard.

---

## 🚨 Problem Statement

Cities often suffer from delayed reporting and resolution of civic issues due to lack of a simple, unified platform. Citizens have no easy way to report problems, and authorities lack real-time visibility.

---

## 💡 Solution

Community Hero App bridges this gap by allowing users to:

- Report civic issues with images
- Capture GPS location automatically or manually
- Categorize problems (pothole, garbage, etc.)
- View all reports in a centralized dashboard
- Track issue status efficiently

---

## ✨ Key Features

- 📷 Image upload for issue reporting  
- 📍 GPS-based location tagging  
- 🗂️ Issue categorization system  
- 📊 Interactive dashboard for monitoring issues  
- 🗺️ Leaflet.js map integration  
- 🗄️ MongoDB database storage  
- 📱 Responsive UI for all devices
